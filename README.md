Enterprise Hardware Catalog - Data Pipeline:
End-to-end pipeline for cleaning, transforming, and analyzing a hardware dataset using Python (Pandas) and SQL (DuckDB).

Structure:
notebook.ipynb – Main pipeline (cleaning, feature engineering, SQL, visualization)
hardware_catalog_clean.csv – Cleaned dataset (Task 1)
assignment_output.csv – Enriched dataset with financial metrics (Task 2)
taxonomy_output.csv – Product classification output (Task 3)

Requirements:
pip install pandas numpy matplotlib duckdb
